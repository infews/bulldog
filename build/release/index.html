Bulldog

Todo.txt's best HTML friend

This is a single-page HTML view of your todo.txt file. Use the command line interface for capture, editing, and completing tasks. But use Bulldog for a more application-like feel for managing your tasks at a higher level.

Please let us know of any issues or feature requests.

This is an open-source project. All of the code is available on Github.The current project backlog is on Pivotal Tracker.
